# Literature index

The full list of papers that back the model, grouped by the assumption each one
supports. This index is tracked in the repository; the **PDFs and their full-text
Markdown conversions are not** (they are copyrighted publisher/author works, and
this is a public repository - see the copyright note in
[`../README.md`](../README.md)).

To read them, download the freely available versions locally with the fetch
script, then optionally convert them to Markdown for offline study:

```bash
python docs/literature/papers/fetch_papers.py      # -> *.pdf (git-ignored)
python docs/literature/papers/convert_papers.py    # -> fulltext/*.md (git-ignored)
```

Each entry links to the free source used by `fetch_papers.py`. The numeric values
taken from each paper are recorded in
[`../calibration.md`](../calibration.md) (the evidence ledger).

---

> **Why the PDFs/full-text are not committed.** The linked versions are free to
> *download for personal study*, but that does not grant the right to
> *redistribute* them. Committing them to a public repository would be copyright
> infringement, so `.gitignore` excludes `*.pdf`, `*.html`, and `fulltext/`. The
> index above and the fetch/convert scripts let anyone reproduce the corpus
> locally in one command, which is the standard, license-safe approach.
